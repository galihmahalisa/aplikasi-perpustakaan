# ğŸ“š Aplikasi Perpustakaan Cepat (Simple Library App)

Aplikasi manajemen koleksi buku berbasis Front-End murni (Vanilla JavaScript) yang berjalan di browser. Aplikasi ini memanfaatkan **Local Storage** untuk menyimpan data buku, memungkinkan fungsionalitas CRUD (Create, Read, Update, Delete) tanpa memerlukan *server* atau *database* eksternal.

## ğŸš€ Fitur Utama

Aplikasi ini dirancang untuk memudahkan pengguna dalam mengelola status buku yang sedang atau telah dibaca:

* **Pencatatan Buku:** Menambah buku baru (Judul, Penulis, ISBN).
* **Daftar Koleksi:** Menampilkan semua buku dalam bentuk tabel yang mudah dibaca.
* **CRUD Operasi:**
    * **Create:** Menambahkan buku melalui formulir.
    * **Read:** Menampilkan semua data yang tersimpan di Local Storage.
    * **Update:** Mengubah status buku menjadi "Selesai Dibaca" atau "Belum Selesai".
    * **Delete:** Menghapus buku dari koleksi.
* **Pencarian Cepat:** Fitur pencarian berdasarkan Judul, Penulis, atau ISBN.
* **Dashboard Ringkasan:** Menampilkan statistik total buku, buku yang sudah selesai, dan yang belum selesai dibaca.

## ğŸ’» Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan teknologi dasar web (Front-End):

* **HTML5:** Untuk struktur dan semantik halaman web.
* **CSS3:** Untuk penataan gaya (styling) dan tampilan yang responsif.
* **Vanilla JavaScript (ES6+):** Untuk seluruh logika aplikasi, manajemen state, dan manipulasi DOM.
* **Local Storage:** Digunakan sebagai pengganti database untuk menyimpan data secara persisten di sisi browser.
